# CALCULADORA CON EXCEPCIONES 
## Excepciones NumberFormatException
### Instrucciones de Ejecución
El programa requiere 3 argumentos: 
número1 operador número2.

Abra la terminal o consola (CMD) en la carpeta del proyecto y ejecute:

````
javac CalculadoraSinExcepciones.java
javac CalculadoraConExcepciones.java
````
Para compilar y verificar que efectivamente funciona, ponga lo siguiente:
````
java CalculadoraConExcepciones <operando1> <operador> <operador2>
````
